# hex-opacity

Convert opacity percentage into a hex value.

## Install

```bash
npm install -g @mskri/hex-opacity
```

## Usage

```bash
hex-opacity <opacity value>
```

### Examples

```bash
hex-opacity 100
FF

hex-opacity 50
80
```
